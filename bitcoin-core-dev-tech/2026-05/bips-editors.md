- This year the repository has an increased number of submissions compared to
  prior years
- Only one BIP Editor had been regularly active on the repository in the first
  quarter
- There is an open proposal to select a new BIP Editor team on the mailing list
- People that are interested should feel free to start reviewing some BIPs to
  demonstrate
- Generally, the repository is open to anyone to comment, more participation
  from the broader ecosystem would be appreciated
  - Feel free to ping people with relevant expertise to point out BIPs that might
    be of interest to review
